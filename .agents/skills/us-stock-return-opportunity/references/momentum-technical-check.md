# Momentum And Technical Check

Use as entry-timing support only. Technical setup cannot override primary evidence or risk gates.

## Checks

| Area | Constructive | Caution |
|---|---|---|
| Trend | Above major moving averages with higher highs | Below key averages or repeated failed breakouts |
| Relative strength | Outperforming sector and index | Up only because whole sector is hot |
| Volume | Breakout or recovery with volume support | Thin liquidity or fading volume |
| Drawdown recovery | Recovers prior breakdown level | Rally stalls below resistance |
| Volatility | Controlled pullbacks | Gap risk, parabolic rise, option-driven spike |
| Support level | Clear invalidation level nearby | No nearby level to define risk |

## Entry Framing

- Prefer staged entry over all-in.
- Define invalidation before entry.
- If price is extended, say what pullback or consolidation would improve risk/reward.
- If technicals conflict with fundamentals, fundamentals control final label.
