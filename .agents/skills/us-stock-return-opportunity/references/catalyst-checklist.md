# Catalyst Checklist

Use when upside depends on timing, rerating, earnings, or a near-term event.

## Catalyst Types

| Catalyst | Evidence To Verify | Failure Mode |
|---|---|---|
| Earnings beat | Reported revenue, margin, EPS, FCF versus prior guidance | Beat driven by one-time item |
| Guidance raise | Management guidance, backlog, demand commentary | Raise already priced in |
| Product cycle | Launch timing, adoption, supply, pricing | Hype without revenue conversion |
| AI or capex cycle | Customer spend, orders, utilization, unit economics | Capex boom without returns |
| Rate or macro shift | Rates, FX, commodity, consumer demand | Macro benefit too broad or temporary |
| Regulation | Approval, subsidy, restriction, litigation outcome | Binary outcome or delayed enforcement |
| Capital return | Buyback, dividend, debt paydown | Buyback funded by leverage or weak FCF |
| Index/flow effect | Inclusion, ETF demand, liquidity change | Flow effect temporary |

## Required Output

- Name the catalyst.
- State expected timing.
- State what primary evidence confirms it.
- State what would disconfirm it.
- Separate "can move the stock" from "improves intrinsic value."
